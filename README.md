🌞 Explorando Desafios e Criando Magia: Miragem Solar 🌙

Desafiei-me a criar algo único e surpreendente, e o resultado é a "Miragem Solar"! Este projeto transforma a experiência de rolar a tela em uma jornada entre dia e noite. Cada transição é uma dança de cores e atmosfera.
Desafio Aceito: Transformar uma ideia em realidade, explorando HTML, CSS e JavaScript para criar uma miragem digital. A jornada foi tão cativante quanto o resultado final.
